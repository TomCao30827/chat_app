Các bước để chạy chương trình:
	1. Khởi chạy MySQL
	2. Tạo database mới và đặt tên là chat_app
	3. Khởi chạy file query chat_app.sql trong thư mục db
	4. Chạy project chat_app và server bằng Netbean
	5. Chạy Main trong project server sau đó chạy Main trong project chat_applicaton và sử dụng ứng dụng chat
